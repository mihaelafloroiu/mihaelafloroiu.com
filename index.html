Skillmeter

Web Design

Skillmeter est un outil qui permet aux entreprises de mesurer les compétences techniques de leurs candidats.

Les recruteurs peuvent utiliser les tests qui sont déjà créés, ou ils peuvent également créer des nouveaux tests qui correspondent mieux à leurs spécificités.

Le but est d'aider les recruteurs à économiser de temps et d’argent mais aussi de fournir une bonne expérience pour les candidates.
